# ai-custdev-cli
Консольное приложение для проведения массовых глубинных интервью с синтетическими пользователями (AI-персонами). Система позволяет загрузить конфигурацию проекта (JSON) и автоматически провести сотни уникальных диалогов между двумя LLM: одна играет роль Интервьюера (исследователя), вторая — Респондента (пользователя).
