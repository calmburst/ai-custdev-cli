# Changelog

Все заметные изменения будут отражаться здесь.

## [0.1.0] - 2025-12-25
### Added
- CLI с полным циклом (генерация персон, интервью, аналитика, советы).
- UX запуска: оценки запросов/токенов/времени, подтверждение, прогресс.
- Режимы именования запусков, теги, отдельные папки вывода.
- Поддержка языка `settings.lang` (default `ru`) и флага `--lang`.
- Аналитика с усиленным промптом и fallback-моделями.
- Конфиг `config/fallback-models.json` с топ-5 бесплатных моделей по умолчанию.
- Demo конфиг `demo_cooking` и эталонные артефакты.
- Документация: README, QUICKSTART, RUN_GUIDE.
